A complete full-stack multi-vendor eCommerce platform built with **Angular 21** (frontend) and **FastAPI** (backend) with **SQLite** database. This platform allows multiple vendors to sell their products, customers to browse and purchase, and admins to manage everything.

## 📌 Project Status

✅ **Completed** - Internship Task 1 for Apexify Technologies

## ✨ Features

### 👤 Customer Features
- User registration & login with JWT authentication
- Browse products with category and price filters
- Product search functionality
- View product details with ratings & reviews
- Add/remove products to shopping cart
- Update cart quantities
- Checkout with shipping address
- Order history tracking
- Submit product reviews & ratings

### 🏪 Vendor Features
- Vendor registration & dashboard
- Add/Edit/Delete products
- Manage product inventory (stock, price)
- View orders containing their products
- Update order status
- Track sales analytics

### 👑 Admin Features
- Complete user management (view, block, role change)
- Product management across all vendors
- Order management for entire platform
- Platform analytics dashboard
- User activity monitoring

### 🔧 Technical Features
- JWT authentication with refresh tokens
- Role-based access control (Customer/Vendor/Admin)
- RESTful API design
- SQLite database with SQLAlchemy ORM
- Email notifications for orders
- Responsive mobile-first design
